EduGenz Admission Follow-Up Tracker is a lightweight and easy-to-use tool designed for schools to manage admission enquiries efficiently.

Features:
✔ Add and manage admission enquiries
✔ Track follow-up dates
✔ Update admission status
✔ Search and filter enquiries
✔ Export data to Excel
✔ Local data storage
✔ Light and Dark mode support
✔ Mobile-friendly interface
